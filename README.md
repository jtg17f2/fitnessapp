## Bug
if you get GoogleSignIn error or the AppAuth error the main solutions would be...
  - first quit xcode 
  - repoen and run again
OR
  - open terminal
  - navigate to fitApp
  - run '$ pod update' 
  OR
  - run '$ pod deintegrate'
  - run '$ pod install'
